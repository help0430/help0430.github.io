建立 Azure CosmosDB

在 Azure 控制頁面建立Azure CosmosDB，如下圖

接著在新增帳戶頁面設定相關資訊

ID: 必須是全站唯一的名稱，只能用小寫英文字母、數字以及-(減號)
API: 設定為 SQL，這會建立 Document 類型的 DB
定用帳戶: 選擇自己的帳戶

位置: 這裡我選美西

然後就按建立，等一下 Azure 就會建立好 CosmosDB 了

使用 Microsoft.Azure.DocumentDB.Core

我們用 Asp.Net Core 建立一組簡單的API來測試 Azure CosmosDB。
開啟 VisualStudio 2017，直接建立 Asp.Net Core 2.1 API 專案，這裡就不多說了。
因為我們是 Asp.Net Core 專案，所以在 NuGet 搜尋 Microsoft.Azure.DocumentDB.Core 並安裝。

建立 DB 和 Collection

要存 Document 之前，要先建立 DB 然後建立 Collection，最後才是 Document，
我們接連呼叫Web api
api/Doc/CreateDB/demo-db
api/Doc/CreateCollection/demo-db/people
用來建立 DB 和 Collection，
在建立 DB 和 Collection 的時候， Id 一定要給，並且 Id 不能重複。
