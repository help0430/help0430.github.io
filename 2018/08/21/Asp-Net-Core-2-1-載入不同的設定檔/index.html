Asp.Net Core 2.1 預設會載入 appsettings.json 和根據環境變數不同而載入 appsettings.{ASPNETCORE_ENVIRONMENT}.json 等設定檔。
但是有時候我們會想要另外載入不同的設定檔，
例如根據 appsettings.json 裡面的設定而決定要載入其他那些設定等狀況，
這時候我們就要自己手動載入了。
